Klasifikasi Text Aduan

Identifikasi apakah teks mengandung aduan/keluhan menggunakan model ML dan rule-based

Pilihan Model Klasifikasi
SVM + Rules Keyword: Mode paling stabil untuk deteksi aduan (berdasarkan evaluasi terbaru)
SVM + Rules Keyword Strict: Lebih ketat, cocok untuk menekan false positive aduan
SVM Only: Hanya probabilitas SVM tanpa koreksi keyword
Enhanced ML + Rule-Based: Kombinasi model lama berbasis RandomForest + aturan
Rule-Based Only: Hanya menggunakan aturan keyword matching
ML Only: Hanya menggunakan model machine learning

Proses Klasifikasi Aduan

System akan mengklasifikasikan teks ke dalam 3 kategori:

  • Aduan Text: Teks yang berisi keluhan/masalah
  • Not Aduan Text: Teks biasa/tidak berisi keluhan
  • Review Needed: Perlu peninjauan manual (ambigu)
